# 这是什么？

这是发生在一座寺庙里的对话，对话的人是一对叫澈丹和空舟的师徒...

内容取自于《扯经》，作者微博@自扯自蛋

我很喜欢这本书。受unix工具fortune的启发，也将这本书的片段制作成了这个小工具，取名nonsense。

为了区分对话，还给对话加了点颜色。
(请原谅我对颜色的审美 :P)

# 使用方式

安装Python3后执行`./nonsense.py`即可。也可以使用`python nonsense.py`执行。

可以把它加到你的PATH路径中，代码写累了就来一段放松放松吧
